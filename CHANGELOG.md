# Changelog

## 0.1.0

Initial public release of `GlmOCRSwift`.

### Summary

- Strict Swift-only and local-only port of the Python GLM-OCR SDK (`https://github.com/zai-org/GLM-OCR`).
- Removed Python parity tooling/docs/tests and baseline artifacts from the repository.
- Added lean non-parity test suite and updated package test target wiring.
- Published package-level docs for Swift-only behavior and supported model/runtime assumptions.
- Added Apache 2.0 licensing scaffolding for this project and third-party legal notices.

### Status

- Swift-only execution model is the only supported path.
- No Python runtime or external parity environment is required.
- Package versioning uses the `v0.1.0` git tag.
